# 📌 Project Overview

The **Admin Panel** is a web-based dashboard developed using **Node.js, Express.js, MongoDB, and EJS**. It provides administrators with an easy-to-use interface for managing website data, users, and content through a secure authentication system.

---

# ✨ Features

- 🔐 Admin Login Authentication
- 📊 Dashboard Overview
- 👤 User Management
- 📝 Content Management
- 📤 Image Upload (Multer)
- 📧 Email Support (Nodemailer)
- 💾 MongoDB Database Integration
- 🔒 Session & Authentication using Passport.js
- 🎨 Responsive User Interface
